# docker-spress

Docker compose configurations to run Spress, PHP static website generator

## how to run

~~~
$ git co https://github.com/akky/docker-spress.git docker-spress
$ cd docker-spress
$ docker-compose up
~~~

Then you may access http://(docker container ip)/ on your browser.

If you do not know the container ip address, try,

~~~
$ docker-machine ls
~~~

and see the 'default' machine's address.
